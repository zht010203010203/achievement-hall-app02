# 图标图片准备说明

## 需要的图片文件

请将以下图片放入 `assets/icons/` 目录：

### 底部导航栏图标
1. **home.png** - 主页/殿堂 (推荐尺寸: 64x64px)
2. **record.png** - 刷题记录 (推荐尺寸: 64x64px)
3. **stats.png** - 统计图表 (推荐尺寸: 64x64px)
4. **achievement.png** - 成就奖杯 (推荐尺寸: 64x64px)
5. **ai.png** - AI助手 (推荐尺寸: 64x64px)
6. **settings.png** - 设置齿轮 (推荐尺寸: 64x64px)

### 功能按钮图标
7. **edit.png** - 编辑/铅笔 (推荐尺寸: 48x48px)
8. **delete.png** - 删除/垃圾桶 (推荐尺寸: 48x48px)

## 图片要求

- **格式**: PNG (支持透明背景)
- **颜色**: 建议使用灰色或蓝色主题色
- **样式**: 简约线条图标，与你提供的"殿堂"图标风格一致
- **背景**: 透明背景

## 临时方案

如果暂时没有图片：
- 应用会自动使用 Material Design 图标作为临时显示
- 中文文字正常显示
- 准备好图片后，放入对应位置即可自动加载

## 图片来源建议

1. **免费图标网站**:
   - https://www.flaticon.com/ (需注明出处)
   - https://icons8.com/ (有免费选项)
   - https://www.iconfinder.com/ (部分免费)

2. **自己设计**:
   - 使用 Figma、Sketch 等设计工具
   - 参考你提供的"殿堂"图标风格统一设计

3. **AI生成**:
   - 使用 DALL-E、Midjourney 等工具生成
